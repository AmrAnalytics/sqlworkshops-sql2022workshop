# Workshop: SQL Server 2022 Workshop

#### <i>A Microsoft workshop from the SQL Server team</i>

<p style="border-bottom: 1px solid lightgrey;"></p>

<h2><img style="float: left; margin: 0px 15px 15px 0px;" src="https://github.com/microsoft/sqlworkshops/blob/master/graphics/textbubble.png?raw=true"><b>     SQL Server 2022 Built-in Query Intelligence</b></h2>

SQL Server 2022 includes built-in query intelligence to gain you performance with no code changes:

- Enhancements to the query store
- Upgrade to SQL Server 2022 with any dbcompat level and get approximate percentile functions and optimized plan forcing.
- Upgrade to SQL Server 2022 with dbcompat 140+ and get memory grant feedback enhancements.
- Upgrade to SQL Server 2022 and get parameter sensitive plan optimization, ce feedback, and dop feedback.

You will cover the following topics in this Module:
